# 🛠 Установка / Развертка

## 1️⃣ Создайте проект Apps Script

1.  Откройте Google Sheet
2.  Меню: **Extensions → Apps Script**
3.  Удалите содержимое стандартного файла `Code.gs`

***

## 2️⃣ Добавьте файлы

### 📄 Code.js

*   **+ → Скрипт**
*   Назовите: `Code.js`
*   Вставьте содержимое вашего `Code.js`

### 📄 ArchiveDialog.html

*   **+ → HTML**
*   Назовите: `ArchiveDialog.html`
*   Вставьте содержимое одноимённого файла

***

## 3️⃣ Запустите и дайте разрешения

1.  В списке функций выберите **onOpen**
2.  Нажмите ▶ **Run**
3.  Подтвердите доступы

После обновления таблицы появится меню:

    Архивация → Открыть архиватор v4.3

# ✅ Использование

## 1. Выбор листа

*   Выберите лист
*   Нажмите **↻ Загрузить**
*   Отобразится структура таблицы

***

## 2. Добавление фильтров

*   Нажмите **+ Добавить фильтр**
*   Настройте:
    *   столбец
    *   оператор
    *   значение
    *   исключить пустые
*   Типы данных:
    *   text
    *   number
    *   date
    *   boolean
    *   списки значений (автоподбор)

***

## 3. Предпросмотр

Нажмите **🔍 Предпросмотр** — появятся два блока:

*   🗂 строки, которые уйдут в архив
*   ✅ строки, которые останутся

Доступны:

*   пагинация,
*   переключение вида (оба / архив / остаток).

***

## 4. Архивация

1.  Введите комментарий
2.  Введите **ARCHIVE**
3.  Нажмите **📦 Архивировать**

Будет выполнено:

*   перенос строк на `<Лист> - Архив`
*   пересборка исходного листа
*   запись лога
