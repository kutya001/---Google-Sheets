# 🛠 Установка / Развертка

## 1️⃣ Создайте проект Apps Script

1.  Откройте Google Sheet
2.  **Extensions → Apps Script**
3.  Удалите содержимое файла `Code.gs`

***

## 2️⃣ Добавьте файлы

✅ **Code.js**  
✅ **ArchiveDialog.html**

> В Apps Script:  
> **+ → Script** → вставьте Code.js  
> **+ → HTML** → вставьте ArchiveDialog.html

***

## 3️⃣ Запустите и дайте разрешения

1.  Выберите функцию **onOpen**
2.  Нажмите **▶ Run**
3.  Подтвердите разрешения

После обновления таблицы появится меню:

    Архивация → Открыть архиватор v4.3

# ✅ Использование

## 1. Выбор листа

Выберите лист → нажмите **Загрузить**

## 2. Добавление фильтров

*   *   Добавить фильтр
*   Выбрать столбец / оператор / значение
*   Поддерживаются типы:
    *   текст
    *   число
    *   дата
    *   булево
    *   списки значений (автоподбор уникальных)

## 3. Предпросмотр

Нажмите **🔍 Предпросмотр**  
Отобразятся два блока:

*   🗂 уходят в архив
*   ✅ остаются

Поддерживается постраничная навигация.

## 4. Архивация

1.  Введите комментарий
2.  Введите **ARCHIVE**
3.  Нажмите **📦 Архивировать**
